Rightsize your Quarkus application in a few clicks

Instructions

  1. Choose the extensions you need
  2. Click on the Generate button to download the demo.zip file
  3. Unzip the file in a directory of your choice
  4. Run mvn quarkus:run in the unzipped directory
  5. Enjoy !
